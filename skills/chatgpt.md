# ChatGPT 适配

## 能力映射

- 最新公开信息：使用联网搜索或深度研究能力
- PDF、Word、表格、截图：优先读取用户上传文件
- 数据清洗与图表：使用数据分析 / Python
- 多视角审查：执行事实审查与落地审查，但不要称为独立交叉验证
- 外部 API：仅在已配置 Actions、Apps 或其他连接工具时调用

## 执行要求

1. 搜索后为关键事实提供可追溯来源。
2. 涉及最新功能、价格、政策和公司动态时必须核实当前状态。
3. PDF 中的表格、图形或页面布局需要实际查看页面。
4. 文件资料与网络资料冲突时，说明版本、时间和口径。
5. 不要声称普通自检等同于多个独立 Agent 的交叉验证。
6. 未配置外部接口时，不要声称已经调用 TianAPI 等服务。

## 推荐配置

Instructions 放核心工作流；Knowledge 放业务背景、评分规则、自家产品事实表和历史案例。动态竞品事实不要只依赖 Knowledge。
