## 交付前自查

- [ ] 关键事实是否有来源和时间？
- [ ] 是否把品牌宣传误当成效果？
- [ ] 是否把少量评论泛化？
- [ ] 是否把相关性写成因果？
- [ ] 是否机械跟进竞品功能？
- [ ] 我方和竞品是否使用同一标准？
- [ ] 证据不足时是否标记未知？
- [ ] 建议是否有行动、优先级和指标？
- [ ] 是否使用过期或下线信息？
- [ ] 结论是否真正回答决策问题？
