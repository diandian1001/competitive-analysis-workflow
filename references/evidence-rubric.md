# 证据分级规则

| 等级 | 标准 | 示例 |
|---|---|---|
| A | 官方信息、实际体验、可靠定量数据直接支持 | 官方帮助文档、财报、后台数据 |
| B | 多个可信来源相互印证 | 官方页面 + 专业媒体 + 实际体验 |
| C | 单一二手来源或有限用户反馈 | 单篇报道、少量应用商店评论 |
| D | 基于公开信息的合理推断 | 根据页面结构推测运营意图 |

## 禁止项

- 用品牌宣传证明业务效果
- 用少量评论推断全部用户
- 引用无法访问的原文
- 不标时间的动态事实
- 把搜索摘要当成完整证据
- 把模型推断写成事实

## 来源冲突

并列展示不同结论，并检查：时间、统计口径、样本、地区、渠道和利益立场。
